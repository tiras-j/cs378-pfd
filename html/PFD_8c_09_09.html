Populates the graph and priority queue with initial state based on the input file. Each NODE is indexed via it's TASKID to allow constant time access to tasks, and each contains a list of TASKIDs which depend on it, as well as the number of tasks it depends on in NODE.dep_count. Priority queue contains all tasks which have no dependencies.
